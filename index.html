Player Tracking

This project is object tracking system using advanced Multi-Object Tracking (MOT) methods. By revisiting the tracking-by-detection paradigm and enhancing the classic DeepSORT tracker, you have created StrongSORT, which achieves record-setting performance on benchmark datasets like MOT17 and MOT20. Additionally, you have introduced two lightweight algorithms, AFLink and GSI, to further improve tracking accuracy and association processes. The integration of StrongSORT with these algorithms results in StrongSORT++, a top-ranking tracker that outperforms other state-of-the-art methods in terms of key metrics. Stay tuned for the release of the code to enable broader research and applications in the field of object tracking.
